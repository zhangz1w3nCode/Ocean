# 品牌设计规范

> 采用全大写16x20像素网格绘制，每个字母由2x2像素点组成。

## Logo设计

### 像素规格
- **像素单元**: 2x2px
- **字母网格**: 16x20 (8宽x10高像素点)
- **配色**: #f9fafb (纯白) + #1f2937 (深色背景)
- **viewBox**: 125x32 (大Logo)

### 布局规范
- **水平居中**: viewBox 从 125x32 调整为 84x32，紧密包围字母内容
- **垂直居中**: 字母 Y 坐标从 6-18 调整为 8-20，上下留白均匀
- **纯图标展示**: Logo 区域仅显示图标，不显示文字

### Logo 容器圆角规范

两个展示位置的 Logo 容器采用统一的圆角比例设计：

| 位置 | 容器尺寸 | 圆角类名 | 圆角值 | 圆角占比 |
|------|---------|---------|--------|---------|
| 侧边栏 | 48x48px | `rounded-lg` | 8px | 17% |
| 项目选择页 | 96x96px | `rounded-2xl` | 16px | 17% |

**设计原则**：
- 两个位置采用相同的圆角占比（17%），确保视觉一致性
- 圆角值随容器尺寸按比例缩放
- 侧边栏：`w-12 h-12 rounded-lg`
- 项目选择页：`w-24 h-24 rounded-2xl`
- 圆角适中，既有圆角感又不显圆润，呈现现代简洁风格

### Logo 悬浮效果规范

Logo 容器悬浮时与项目卡片、"打开文件夹"按钮保持一致的交互体验：

```tsx
// 统一的悬浮效果
style={{
  transition: 'transform 0.2s, box-shadow 0.2s',
}}
onMouseEnter={(e) => {
  e.currentTarget.style.transform = 'translateY(-2px)'
  e.currentTarget.style.boxShadow = '0 6px 20px rgba(0, 0, 0, 0.5)' // 深色背景用更强阴影
}}
onMouseLeave={(e) => {
  e.currentTarget.style.transform = 'translateY(0)'
  e.currentTarget.style.boxShadow = '0 4px 16px rgba(0, 0, 0, 0.4)' // 恢复默认阴影
}}
```

**悬浮效果对比**：
| 元素 | 上浮 | 阴影变化 | 背景色变化 |
|------|------|---------|-----------|
| Logo（深色背景） | -2px | 增强阴影 | 无 |
| 项目卡片（浅色背景） | -2px | 添加阴影 | 无 |
| 打开文件夹按钮 | -2px | 添加阴影 | 变深 |

**注意**：
- 深色背景的 Logo 在默认状态已有阴影，悬浮时增强阴影效果
- 浅色背景的卡片/按钮默认无阴影，悬浮时添加阴影
- SVG 元素添加 `pointer-events-none` 避免事件穿透问题
